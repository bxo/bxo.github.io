手动安装 Apache Server 2.0.63

官方下载 httpd-2.0.63.tar.gz

1.解压缩httpd-2.0.63.tar.gz到临时目录如~/httpd-2.0.63 

    	
tar -zxvf httpd-2.0.63.tar.gz

2.进入~/httpd-2.0.63目录，执行 

    	
#./configure --prefix=/usr/local/apache2 --enable-module=so 
#make 
#make install

3.启动和停止apache 

    	
#./apachectl start 
#./apachectl start

如果启动时提示: httpd: Could not determine the server's fully qualified domain name, using 127.0.1.1 for ServerName
可以忽略此信息或在http.conf中加一行: ServerName 127.0.0.1:80

如果启动失败，在 logs/error_log 发现如下出错信息
Couldn't set permissions on cross-process lock

检查 conf/httpd.conf 文件中的 User 和 Group 属性值。 Group值有如果是 “#-1”等错误值，确保配置如下 

    	
User nobody
Group nogroup
