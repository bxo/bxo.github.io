Hello World 构建文件

创建一个文件，命名为“build.gradle”。文件内容如下： 

	
task helloWorld << {
	println 'hello, world'
}
执行如下命令将会得到输出如下： 
	
$ gradle -q helloWorld
hello, world

下面我们再试另外一个简单的例子。其中有两个任务和一个依赖，
将build.gradle文件内容改为如下，执行命令可看到输出上面相同的内容。 

	
task hello << {
print 'hello, '
}
task world(dependsOn: hello) << {
println 'world'
}
我们执行第二个任务，结果是输出相同的内容。 
	
$ gradle -q world
hello, world
